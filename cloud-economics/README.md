# AWS SimuLearn: Cloud Economics

## Overview
This guided AWS lab focused on cloud cost awareness and infrastructure rightsizing. The exercise introduced the relationship between instance sizing and overall cloud spending, using a pricing estimate scenario in AWS.

## Objective
To understand how infrastructure choices affect cloud costs and to apply a simple rightsizing action in a cloud pricing estimate.

## Tasks Performed
- Reviewed an AWS pricing estimate
- Changed the EC2 instance type from the **t3** family to **t2.micro**
- Generated a new pricing estimate link after the change
- Validated that the estimate reflected the updated instance size

## AWS Services and Tools Involved
- Amazon EC2
- AWS Pricing Calculator

## What I Learned
- How EC2 instance type selection affects estimated cloud cost
- Why rightsizing is important in cloud environments
- How pricing estimation supports cost-conscious decision-making
- The value of reviewing infrastructure choices before deployment

## Business Relevance
In enterprise cloud environments, choosing the right instance type is important for balancing performance and cost. This lab introduced a basic FinOps mindset by showing how unnecessary cloud spend can be reduced through better sizing decisions.

- Use of AWS pricing estimation tools
