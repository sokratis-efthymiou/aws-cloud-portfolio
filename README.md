# AWS Portfolio

This repository includes real AWS projects, completed guided labs, and hands-on cloud portfolio work.

## Repository Scope
This portfolio is being built progressively and will include:
- real AWS projects
- completed guided labs
- hands-on rebuilds
- certificates and evidence of completion
- structured documentation of learning outcomes

## Current Topics
- Cloud economics
- Networking concepts
- Compute
- Databases
- Security and governance

## Evidence Approach
For guided labs completed in temporary environments, I include completion certificates and written summaries.

For independently rebuilt projects, I include screenshots, diagrams, and structured documentation.

## Current Status
This repository starts with completed AWS labs and will expand into more independent hands-on AWS projects over time.
